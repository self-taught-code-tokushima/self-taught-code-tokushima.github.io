GitHub Copilot を Visual Studio に導入する

GitHub Copilot Free を有効にする​

以下のどちらかにアクセスすることで GitHub Copilot Free を有効化できます。

  • https://github.com/settings/copilot
    • 有効化するリンクがあります
  • https://github.com/copilot

Visual Studio からも直接有効化することもできます。

  • キーチェーンに GitHub アカウントを追加する - Visual Studio (Windows) | Microsoft Learn

以下にも参考情報があります。

  • Copilot プランの概要 - GitHub Docs

Visual Studio で利用する​

  • Visual Studio での GitHub Copilot Free - Visual Studio (Windows) | Microsoft Learn

基本的には、GitHub Copilot の機能を有効化し、GitHub アカウントでログインするだけです。

2022 では右上にアイコンがあります。

制限について​

  • 個々の Copilot プランと利点について - GitHub Docs
  • IDE 内で毎月最大 2,000 コード補完を使用できます
  • Copilot Chat 内で毎月最大 50 合計チャット リクエストを使用できます
