Unity Editor の設定

ここではやっておくべき Unity Editor の設定について説明します。

再生モード開始時設定の変更​

  • Unity 6
  • Unity 2022

この設定を Do not reload Domain or Scene に設定します。

再生モードの開始時オプションは有効にしますが、再ロードはどちらもオフにしておきます。

この設定をしていないと、ゲームのデバッグ実行の開始のたびにリロードが発生してしまいます。
